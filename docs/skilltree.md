# 스킬트리 (시즌1 / 만렙 20)

> 설계 원본. `src/data/skilltree.ts`가 이 구조를 그대로 데이터화한 것이다.
> 새 스킬 추가는 데이터 한 줄. 수치는 Phase 1 placeholder.

## 구조 규칙

- 레벨 = **각 직업 트리마다 독립 지급되는 스킬포인트**. Lv20 → 세 직업에 각각 20pt.
- 직업당 **3갈래**. 갈래마다 **13pt** (스킬 12 + 궁극기 1).
- 갈래 표준 형태:
  - **T1** (게이트 없음): 스킬 4 × 2랭크 = 8pt
  - **T2** (게이트: 갈래 5pt): 스킬 2 × 2랭크 = 4pt
  - **T3 궁극기** (게이트: 갈래 12pt): 1 × 1랭크 = 1pt
- 게이트: 중단(T2) = 갈래 누적 `GATE_T2`(5) / 궁극기(T3) = 갈래 누적 `GATE_T3`(12).
- **단단한 칸막이**: 만렙 20pt로는 한 갈래에 12pt를 쏟아야 궁극기 1개 →
  **직업당 궁극기는 하나만**. 남은 8pt로는 다른 궁극기(게이트 12) 불가.
- 삼각 상성 천장은 스킬 풀업으로도 안 깨진다.
